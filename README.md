## Overview

Celery Executor is a native Windows utility for injecting and running Lua scripts within the Roblox client. Its interface is intentionally straightforward, giving users a direct way to load custom scripts without recurring key generation or a paid subscription.

A one-click injection workflow and a persistent script queue form the core of the application. Celery can be used for repetitive task automation or custom-script testing while maintaining a responsive execution environment on Windows.

## Frequently Asked Questions

**Can Celery Executor be considered safe?**  
The software is supplied as-is, so use it at your own discretion. Review the Roblox terms of service and the policies that apply to third-party software before using it.

**Is the current Roblox release supported?**  
Celery's update engine is intended to keep the executor compatible with current Roblox versions. When a Roblox update causes an issue, a corrective release is generally made within hours.

**What does Celery provide compared with paid executors?**  
Celery is a free executor with no key requirement and covers the main functionality without a subscription. Paid alternatives may include extra support or scripts that are not included here.

**Can using Celery result in a ban?**  
Third-party Roblox tools carry account-related risks, and this project makes no promise against enforcement. Use the software carefully and accept those risks before proceeding.

**Where does the application keep downloaded scripts?**  
Downloaded files are placed in the `script_hub/` directory inside the installation folder. Scripts can also be added or deleted manually from that location.
